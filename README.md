# NeuroStack

Think Better. Live Smarter.

A complete thinking system built from cognitive science, systems thinking, and first principles.

## Features

- **8 Thinking Frameworks**: Step-by-step systems for decision-making, first principles reasoning, habit change, bias reduction, and more
- **12 Deep Articles**: Comprehensive insights on brain & behavior, AI & thinking, systems thinking, biases, identity, and mental models
- **6 Interactive Tools**: Practice frameworks with cognitive load meter, mental model recommender, habit analyzer, and more
- **Knowledge Graph**: Visual web connecting all frameworks, concepts, and articles
- **Modern UI**: Ultra-clean, premium design with smooth animations and micro-interactions
